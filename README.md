2. Programma raadplegen

	- hoe scrollen tussen dagen?
		- horizontaal?
		- animaties?
4. Foto's bekijken
	- hoe organiseerd? Per dag?
	- javascript foto-browser?

		- hoeveel per pagina?
		- animaties?
		- ...?

	- beveiliging tegen te veel entries van 1 IP?
